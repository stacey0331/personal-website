## Local Installation

1. Run the server

    ```
    cd server
    npm i
    localhost:8000
    ```

2. In another window, run the client

    ```
    cd client
    npm i
    localhost:3000
    ```

3. To see all the form responses, run the admin

    ```
    cd admin
    npm i
    localhost:3001
    ```

** Note: these are default port numbers. It might change if the port is already in use on your computer.
