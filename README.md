## Local Installation

1. Download all the packages and the dependencies

    ```
    cd client
    npm i
    cd ..
    ```

    ```
    cd server
    npm i
    cd ..
    ```
    ```
    #! Run this only when you're the admin
    cd admin
    npm i
    cd ..
    ```

2. Run ```npm start``` inside all folders, and now you can see the website running: 

    Frontend(client)
    ```
    localhost:3000
    ```

    Admin
    ```
    localhost:3001
    ```
